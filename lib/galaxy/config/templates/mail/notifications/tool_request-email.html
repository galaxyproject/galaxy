Use this template to customize the HTML email your admins will receive when a user submits a tool installation request. Copy the file to {{ templates_dir }}/mail/notifications/tool_request-email.html and modify as required. If you are adding URLs, remember that only absolute URLs (with a domain name) make sense in email! They can be served from any stable location, including your Galaxy server or GitHub. The following variables are available for inserting into the HTML with Jinja2 syntax, like {{ variable_name }}.
